# Pengertian 

*Software Define Radio* (SDR) adalah sistem komunikasi radio,  menggunakan perangkat lunak untuk membuat sistem radio. **perbedaan** utama software define radio dengan radio tradisional adalah perangkat yang tadinya ada pada hardware di gantikan dengan software. perangkat yang digantikan dengan software seperti (misalnya mixer, filter, amplifier, modulator / demodulator, detektor, dll.). 


# Cara kerja 
Software define radio berkerja dengan menggunakan gelombang digital, yang dikenal dengan digital signal prosesing. konsep utama dari (SDR) adalah mengubah gelombang suara pada sound card komputer, menjadi gelombang digital atau analog. 

# Manfaat dari software degine radio

ada banyak sekali manfaat yang bisa kita dapatkan dari sofware define radio. aplikasi yang dapat dibuat sebagai berikut:

* Radar ADSB
* Femtocell
* Pemancar radio 
* Penerima radio
* Pemancar stasiun TV
* Pemancar stasiun radio
* Pager 
* Radio penerima satelit

dan masih banyak lagi, manfaat yang kita buat. 

# Belajar dari mana?

Untuk memulai belajar SDR sekiranya seorang engineer harus mengusai berikut ini: 

* Sistem block komunikasi 
* Pemerograman python atau C
* Sistem operasi berbasis open source
* Antenna dan propagasi
* Sistem komunikasi radio
* FPGA
* Embeded linux


Kenapa itu harus dipelajari?, karena dalam satu sama lain saling berkaitan. tidak bisa tidak dikuasa secara parsial. tidak perlu dihapal cukup dipahami dan dimengerti.
